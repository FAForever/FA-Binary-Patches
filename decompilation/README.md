# Decompilation Documentation

This directory contains decompiled C++ code and architectural findings of the `MohoEngine.dll` and other game binaries.

## Methodology

The decompilation process is highly experimental and utilizes AI and the **MCP Ghidra server**. 

**Disclaimer & Warning:** 
All decompiled code, signatures, and logic presented here must be manually reviewed and verified. Since the code is AI-generated and structurally inferred from Ghidra decompilation, it may contain inaccuracies or incomplete logic.

## Purpose

Despite the experimental nature of this decompilation, the findings are intended to assist in the future development of **Forged Alliance Forever (FAF)**. By understanding the game's internal mechanics—such as session management, networking, and replay functionality—these insights could help with implementing new features like the rejoin functionality and improving the overall stability of the engine.
